Scanning Using Nikto
Nikto is an open-source web server vulnerability scanner that identifies security issues, misconfigurations, and outdated software. It’s pre-installed on Kali Linux and scans for over 6,700 vulnerabilities. It’s noisy, so always get permission before scanning to avoid legal issues.
Steps to Use Nikto on Kali Linux
Verify Nikto Installation
Nikto comes pre-installed on Kali Linux.
Check version:
nikto -Version
Update Nikto Database
Update vulnerability database:
nikto -update
Get Authorization
Obtain explicit permission from the target system’s owner before scanning.
Run a Basic Scan
Scan a target website:
nikto -h http://example.com
Common Scan Options
HTTPS Scan:
nikto -h https://example.com -ssl
Specific Port:
nikto -h example.com -p 8080
Save Output (HTML format):
nikto -h example.com -o scan.html -Format htm
Review Results
Check output for vulnerabilities (e.g., outdated software, misconfigurations).
Example output:
+ Server: Apache/2.4.46 (Unix)
+ Outdated software detected: Apache/2.4.46
+ Directory indexing found at: /images/
Prioritize high-severity issues and verify manually to avoid false positives.
